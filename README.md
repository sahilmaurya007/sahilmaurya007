## 📂 Key Projects Detail

### 1. Quantum-Resistant Hardware Accelerator (Kyber/Dilithium)
* **Role:** Lead Architect
* **Tech:** SystemVerilog, ASAP7nm, Vivado
* **Description:** Developed a specialized hardware unit for the Number Theoretic Transform (NTT), reducing the computational latency of Lattice-based PQC schemes by 40% compared to software implementations.

### 2. Side-Channel Aware RISC-V Pipeline
* **Role:** Security Researcher
* **Tech:** RISC-V, RTL Synthesis, Chipwhisperer
* **Description:** Integrated randomization and masking techniques at the register-transfer level to prevent Differential Power Analysis (DPA) on AES-128 cryptographic modules.

### 3. OpenSource ASIC Flow (RTL-to-GDSII)
* **Role:** Physical Design Engineer
* **Tech:** SkyWater 130nm, OpenLane, Magic
* **Description:** Successfully completed a full digital flow for a custom processor, achieving 98.5% cell density with zero LVS/DRC violations.
