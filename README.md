## 🚀 Research Projects & Implementations

### **Project 1: Kyber-Accel-SoC (PQC Coprocessor)**
<p align="center">
  <img src="https://images.unsplash.com/photo-1509228468518-180dd482180c?ixlib=rb-1.2.1&auto=format&fit=crop&w=400&q=80" align="left" width="200" style="margin-right: 20px; border-radius: 10px;" />
  <br>
  <b>Lattice-based PQC Acceleration</b><br>
  Designed a hardware NTT accelerator interfaced with a 5-stage RISC-V pipeline via AXI4-Lite. Achieved 12.5x speedup for Kyber-768 Key Encapsulation (KEM) vs. software.
</p>

<br>

### **Project 2: AxMM-RLWE (Approximate Multiplier)**
<p align="center">
  <img src="https://images.unsplash.com/photo-1518770660439-4636190af475?ixlib=rb-1.2.1&auto=format&fit=crop&w=400&q=80" align="right" width="200" style="margin-left: 20px; border-radius: 10px;" />
  <br>
  <b>Low-Power VLSI for Cryptography</b><br>
  Implemented an Error-Configurable Approximate Modular Multiplier (AxMM) for Ring-LWE. 22% reduction in LUT usage and 15% better Area-Time-Product (ATP) on 45nm ASIC.
</p>
