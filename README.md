 ## 💻 Hardware-Software Interface (HSI) Snippet 
  
 Example C-header for RISC-V to communicate with the PQC accelerator: 
  
 ```c 
 #define PQC_BASE_ADDR 0x40000000 
 #define PQC_CTRL      (*(volatile uint32_t*)(PQC_BASE_ADDR + 0x00)) 
 #define PQC_STATUS    (*(volatile uint32_t*)(PQC_BASE_ADDR + 0x04)) 
 #define PQC_MEM_START (volatile uint32_t*)(PQC_BASE_ADDR + 0x1000) 
  
 void trigger_pqc_operation() { 
     while(PQC_STATUS & 0x01); // Wait if busy 
     PQC_CTRL = 0x01;          // Start NTT 
     while(!(PQC_STATUS & 0x02)); // Wait for Done 
 } 
 ```
